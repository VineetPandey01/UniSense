# UniSense – Learning Application for Deaf and Blind People

UniSense is an accessibility-focused mobile application designed to support learning and communication for people with sensory impairments. The application integrates speech processing, gesture interaction, and tactile feedback to create an inclusive learning environment for deaf, blind, and deaf-blind users.

The project aims to reduce accessibility barriers in digital education by providing multiple sensory interaction modes within a single platform.

---

## Features

- Speech-to-Text conversion for deaf users
- Text-to-Speech support for blind users
- Sign language recognition using camera input
- Haptic feedback for tactile interaction
- Braille learning modules
- Braille testing and practice
- Voice-based navigation commands
- Accessible UI designed with Jetpack Compose
- Multisensory interaction support

---

## App Modules

### 1. Blind Assistance Module
Provides an accessible interface for visually impaired users with voice guidance and text-to-speech navigation.

Functions:
- Learn Braille Letters
- Braille Test
- Braille Menu
- Voice command navigation
- SOS support

### 2. Deaf Assistance Module
Supports deaf users with visual content, captions, and sign language recognition.

Functions:
- Sign language lessons
- Gesture recognition
- Caption-based communication

### 3. Accessibility Engine
Manages user sensory profiles and dynamically adapts the interface.

Functions:
- Speech-to-text
- Text-to-speech
- Haptic feedback
- Multisensory UI adaptation
